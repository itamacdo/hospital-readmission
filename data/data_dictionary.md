# Dicionário de Dados - Saúde

| Variável | Descrição | Tipo |
| :--- | :--- | :--- |
| `idade` | Idade do paciente no momento da internação | Numérico |
| `emergencias` | Quantidade de vezes que o paciente usou o PS no último ano | Numérico |
| `diabetes` | Flag se o paciente possui diagnóstico crônico de diabetes | Binário (0/1) |
| `readmitido` | Variável alvo: se o paciente retornou em 30 dias | Binário (Target) |
